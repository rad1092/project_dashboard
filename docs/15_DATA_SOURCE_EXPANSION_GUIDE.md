# 데이터 소스 확장 가이드

## 현재 상태

- 전처리 완료 CSV 사용
- 외부 폴더 읽기 전용
- 실시간 공공 API 미연결

## 실시간 특보로 확장할 때

1. `dashboard/services/disaster_data.py` 에 실시간 로딩 함수 추가
2. 기존 `build_alert_summary()` 와 같은 반환 형식을 유지
3. `pages/2_대피소_추천.py` 는 같은 인터페이스로 결과만 받아 쓰게 유지
4. `pages/4_실시간_준비.py` 의 스텁과 docs 를 실제 구현 기준으로 갱신

## 자동 위치로 확장할 때

1. 페이지 입력 구조는 유지
2. 수동 좌표 입력 대신 브라우저 권한 결과를 세션 상태에 채움
3. 좌표가 들어간 뒤 추천 함수는 그대로 재사용

## 경로 API 로 확장할 때

1. 현재 직선 거리 계산과 별도 함수로 분리
2. 추천 정렬 기준이 바뀌는지 먼저 결정
3. 유료 API 사용 여부와 비용 정책을 문서화
