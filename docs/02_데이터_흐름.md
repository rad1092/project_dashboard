## 화면별 사용 방식

### `app.py`

- `HOME`에서 공통 데이터 요약과 화면 개요를 보여준다.

### `pages/1_대피_안내_시뮬레이션.py`

- 정적 특보/대피소 CSV를 기준으로 현재 위치와 선택 재난 유형에 맞는 대피 경로를 시뮬레이션한다.

### `pages/2_실시간_대피_안내.py`

- 최신 재난문자 CSV를 읽고 감지 지역 기준 최신 문자 1건을 자동 선택해 대피소 추천과 경로 안내를 만든다.

### `pages/3_데이터_분석.py`

- 특보 이력 데이터를 차트와 원본 데이터 표로 재구성한다.

### `pages/4_권역_대피소_지도.py`

- 저장된 HTML 지도를 그대로 불러와 전체 대피소 위치를 보여준다.
