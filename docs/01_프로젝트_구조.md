# 프로젝트 구조

## 주요 파일

- `app.py`
  - `st.navigation` 기반 라우터와 `HOME` 화면을 관리한다.
  - 공통 페이지 설정, 제목, 섹션 helper도 함께 제공한다.
- `dashboard_data.py`
  - 전처리 CSV를 읽고 공통 데이터프레임 형태로 정리한다.
- `pages/1_대피_안내_시뮬레이션.py`
  - `대피 안내 시뮬레이션` 화면
- `pages/2_실시간_대피_안내.py`
  - `실시간 대피 안내` 화면
- `pages/3_데이터_분석.py`
  - `데이터 분석` 화면
- `pages/4_권역_대피소_지도.py`
  - `권역 대피소 지도` 화면
