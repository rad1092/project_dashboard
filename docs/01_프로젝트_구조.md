# 프로젝트 구조

## 주요 파일

- `app.py`
  - `st.navigation` 기반 메인 엔트리포인트
  - HOME 랜딩 화면 구성
  - 공용 페이지 스타일 helper와 전처리 CSV 로더 제공
- `pages/1_대피_안내_시뮬레이션.py`
  - 선택한 재난 기준의 대피 경로 시뮬레이션 화면
- `pages/2_실시간_대피_안내.py`
  - 실시간 재난문자 기반 대피소 추천 화면
- `pages/3_데이터_분석.py`
  - 원본 전처리 CSV를 재집계한 분석 화면
  - 권역 대피소 지도 HTML 임베드 탭 포함
- `preprocessing_code/mock_disaster_message.py`
  - 실시간 페이지 시험용 모의 재난문자 CSV 생성기

## 디렉터리 개요

```text
.
|- app.py
|- pages/
|  |- 1_대피_안내_시뮬레이션.py
|  |- 2_실시간_대피_안내.py
|  |- 3_데이터_분석.py
|- preprocessing_code/
|- preprocessing_data/
|- docs/
`- tests/
```
