# 재난 대피 대시보드

전처리된 재난 특보 데이터와 크롤링된 재난문자를 바탕으로 시뮬레이션, 실시간 안내, 분석, 지도 화면을 제공하는 Streamlit 앱이다.

## 현재 화면 구성

- `HOME`
  - 전체 화면 구성을 한 번에 확인하는 시작 화면
- `대피 안내 시뮬레이션`
  - 현재 위치와 선택한 재난 유형 기준으로 대피소와 경로를 시연
- `실시간 대피 안내`
  - 감지 지역 기준 최신 재난문자를 자동 반영해 대피 안내를 생성
- `데이터 분석`
  - 특보 추이, 지역별 분포, 대피소 구성 차트를 확인
- `권역 대피소 지도`
  - 권역 전체 대피소 지도를 한 화면에 표시

## 실행 방법

```powershell
streamlit run app.py
```

## 테스트

```powershell
.\.venv\Scripts\python -m pytest -q
```

테스트는 `PROJECT_DASHBOARD_IMPORT_ONLY=1`을 사용해 페이지 import 시 자동 실행을 막고, 주요 모듈과 helper를 검증한다.
